前端基础|Ajax详解

一、Ajax概述

  • 概述

    Ajax全称为Asynchronous JavaScript And XML，就是异步的JS 和 XML。通过AJAX可以在浏览器中向服务器发送异步请求，最大的优势：页面无刷新获取数据。Ajax不是新的编程语言，而是一种将现有的标准组合在一起使用的新方式。

  • 作用

    • 可以局部刷新页面。
    • 可以发起异步请求。

  • 缺点

    • 没有浏览历史，不能回退
    • 存在跨域问题

  • 同步和异步

二、XMLHttpRequest对象详细介绍

XMLHttpRequest是异步请求对象，用于发送异步请求。

  • 成员变量

    • onreadystatechange
      • 用于指定函数，该函数可以监听异步请求对象状态改变
    • readyState：异步对象状态值，有5种状态值
      • 0 : 异步对象还没创建
      • 1 : 异步对象开始发送请求
      • 2 : 异步对象发送请求完成
      • 3 : 异步对象开始读取服务器发送的响应
      • 4 : 异步对象读取服务器发送的响应完成
    • responseText
      • 用于获取服务器发送的响应正文
    • status : 200
      • 服务器的响应状态码
    • xhr.responseType = ‘json’（在open方法后设置）
      • 设置返回的数据格式为json格式
    • onerror 出错的回调函数
    • ontimeout 请求超时的回调
    • timeout 指定超时时间（单位：毫秒）

  • 成员方法

    • open(type,url,flag)
      • type : 请求方式（get/post）
      • url : 访问路径
      • flag : 当值为true时，就是异步请求，false为同步请求
    • send(请求参数)
      • get请求，请求参数应该在url后面拼接
      • post请求，使用key = value键值传参
    • setRequestHeader(name,value)
      • 设置请求头

  • 创建异步对象

三、Ajax发送get请求

  • 开发步骤
    1. 创建XMLHttpRequest对象
    2. 设置监听XMLHttpRequest对象状态改变
      • 监听异步请求成功，获取响应正文（responseText/response）
    3. 打开连接，配置参数
      • type : ‘get’
      • url : 访问路径
      • flag : true
    4. 设置数据返回格式 xhr.responseType = 'json'
    5. send()发送请求
